# 📊 Fitbit Digital Marketing Analytics Dashboard

Power BI | Power Query | DAX

## 🔹 Project Overview
I developed a Digital Marketing Analytics Dashboard for Fitbit using Power BI to provide clear, actionable insights into campaign performance across ad platforms, product categories, and time periods.



## 🔹 Data Preparation & Modeling
- Cleaned and transformed data using Power Query
- Built a structured data model
- Created key DAX measures:
  - Total Revenue
  - Total Conversions
  - Conversion Rate (CR)
  - Click-Through Rate (CTR)
  - Revenue vs Target

## 🔹 Key Insights
- Total Revenue: $411K across 100 campaigns
- Google & Facebook generate highest revenue
- Peak engagement at specific hours
- High-performing product categories

## 🔹 Business Recommendations
- Focus on Google & Facebook
- Optimize campaign timing
- Invest in top-performing products

## 🔹 Tools Used
- Power BI
- Power Query
- DAX
